* ### 安装unzip

  1.对于 Debian / Ubuntu / Armbian 系统 (最常见):

```Bash
apt-get update && apt-get install -y unzip
```Bash
  2.对于 CentOS / RHEL / Fedora 系统:
```Bash
yum install -y unzip
```
  3.对于 Alpine Linux 系统:
```Bash
apk add unzip
```
